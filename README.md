# hw_getCleanData
Coursera Getting and Cleaning data course hw 4
The scrips analyses the data of the UCI_HAR dataset on a phone's movement data
Before executing change directory to the top level directory of the dataset
Make sure that dplyr is installed
The script combines the test and training data.
It then reports mean and std deviations in the dataframe Extract_measurements
Finally it groups the data and reports averages by subject and activity
